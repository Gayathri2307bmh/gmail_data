# Gmail Data

A powerful Flask-based application that connects to Gmail using Google OAuth and intelligently extracts, filters, and structures email data based on keywords.

##  What this project does

This tool helps you:
- Search Gmail inbox using keywords
- Highlight matched content (subject, sender, preview, body)
- Extract useful data automatically:
  - Emails
  - Phone numbers
  - Dates
  - Amounts
  - Links
- Export results as:
  - JSON
  - CSV

##  Key Features

-  Secure Google OAuth login
-  Gmail API integration (read-only access)
-  Multilingual-friendly keyword matching
-  Smart highlighting of matched keywords
-  Structured data extraction from emails
-  Export functionality (JSON & CSV)
-  Session-based processing (no permanent storage)

##  Tech Stack

- Python
- Flask
- HTML, CSS
- Google Gmail API
- OAuth 2.0
